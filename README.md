# BattleShips

An implementation of the two player game Battleships.

--

## Running the project locally

clone the repo, cd into the cloned directory, `yarn` and then `yarn start`


---

## Implementation phases

Features will be developed in the following phases.

### Phase 1

1. Allow local play against computer
2. Styles of the game are almost finalized with good animations.
3. Max keyboard support wherever possible.
4. Responsive design makes the game playable on mobile


### Phase 2

1. Add an Elxir/Phoenix server that allows online 2-player game using web sockets


### Phase 3

1. Allow the computer to "learn" from games played against it.
2. The computer uses the heatmap/probability data to take its shots and place its ships.
